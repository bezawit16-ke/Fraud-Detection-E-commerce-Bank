# Fraud Detection Project

## Project Overview
This project focuses on detecting fraudulent transactions for e-commerce and banking using machine learning.

## Interim-1 Progress
- **Data Cleaning:** Processed raw transaction data and corrected timestamp formats.
- **EDA:** Analyzed class distribution and identified a ~9% fraud rate.
- **Visualizations:** Created distributions of transactions to identify patterns.
- **Challenge:** Identified significant class imbalance which will be addressed using SMOTE in Task 2.

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Explore the analysis in `notebooks/`.
